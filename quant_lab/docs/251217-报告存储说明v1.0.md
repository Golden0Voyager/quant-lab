## 🎯 报告生成流程

### 1. 运行主程序
```bash
python main.py
```

### 2. 自动创建目录
```python
# 代码逻辑（main.py:163-168）
now = datetime.now()
date_folder = now.strftime('%y%m%d')      # 例如：251217
report_base = "Report"
report_dir = os.path.join(report_base, date_folder)
os.makedirs(report_dir, exist_ok=True)    # 自动创建
```

### 3. 生成报告文件
```python
# 代码逻辑（main.py:170-172）
time_prefix = now.strftime('%H%M%S')      # 例如：093000
filename = os.path.join(report_dir, f"{time_prefix}_Monitor_Report.md")
# 完整路径：Report/251217/093000_Monitor_Report.md
```

### 4. 输出提示
```

📊 统计: 成功 10 | 失败 0
```
