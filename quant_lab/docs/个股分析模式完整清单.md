## 📊 模式维度分析

你的系统支持**两个独立维度**的组合配置：

### 维度1：分析深度 (--analysis-mode)
| 模式 | 说明 | 使用层级 | 速度 | 适用场景 |
|------|------|---------|------|---------|
| **fast** | 快速模式 | 仅Worker层 | ⚡⚡⚡ 最快 | 日常监控、批量扫描 |
| **deep** | 深度模式 | Worker + Brain层 | 🐢 慢 | 重点个股深度研究 |
| **auto** | 智能触发 | 自动判断是否启用Brain | ⚡⚡ 较快 | 平衡效率与深度 |

### 维度2：分析风格 (--prompt-version)
| 风格 | 核心理念 | 输出特点 | 适合人群 |
|------|---------|---------|---------|
| **professional** | 专业分析师 | 机构研报风格，结构化报告 | 有投资经验者 |
| **value_first** | 价值优先 | 价值是锚，技术是择时 | 中长期价值投资者 |
| **quant_hybrid** | 量化混合 | 多因子打分，大白话解释 | 量化投资爱好者 |

**重要说明**：
- `--prompt-version` 仅在使用 **Brain层** 时生效（即 `deep` 或 `auto` 触发时）
- `fast` 模式使用固定的简洁 Prompt，不区分版本

## 🎯 所有可能的组合模式

### 有效组合（共8种）

#### 1️⃣ 快速模式（1种）
```bash
--analysis-mode fast
```
- **说明**: Worker层快速分析，使用简洁Prompt
- **不区分 prompt-version**
- **速度**: 5-10秒/只
- **适用**: Watchlist批量监控

---

#### 2️⃣ 深度模式 - 专业分析师版（默认）
```bash
--analysis-mode deep --prompt-version professional
```
- **输出风格**: 机构研报格式
- **包含**: 综合评级、核心逻辑、操作策略、风险提示
- **速度**: 20-40秒/只
- **适用**: 重点个股深度研究

---

#### 3️⃣ 深度模式 - 价值投资版
```bash
--analysis-mode deep --prompt-version value_first
```
- **输出风格**: 价值+择时视角
- **核心**: 估值锚定 + 技术择时 + 催化剂
- **特点**: 强调安全边际、现金流质量
- **适用**: 中长期价值投资者

---

#### 4️⃣ 深度模式 - 量化评分版
```bash
--analysis-mode deep --prompt-version quant_hybrid
```
- **输出风格**: 多因子打分表 + 大白话解释
- **评分维度**: 估值(30%) + 质量(25%) + 动量(25%) + 情绪(20%)
- **特点**: 系统化、量化、通俗易懂
- **适用**: 量化投资爱好者

---

#### 5️⃣ 智能模式 - 专业分析师版
```bash
--analysis-mode auto --prompt-version professional
```
- **触发机制**: 信号评分 ≥ 3分自动启用Brain层
- **输出**: 有强信号时使用深度分析，否则快速分析
- **速度**: 5-40秒（动态）
- **适用**: 日常监控+重点深挖

---

#### 6️⃣ 智能模式 - 价值投资版
```bash
--analysis-mode auto --prompt-version value_first
```
- **触发机制**: 同上
- **输出**: 价值投资视角的深度分析
- **适用**: 价值投资者的日常监控

---

#### 7️⃣ 智能模式 - 量化评分版
```bash
--analysis-mode auto --prompt-version quant_hybrid
```
- **触发机制**: 同上
- **输出**: 量化打分 + 大白话解释
- **适用**: 量化投资者的日常监控

---

#### 8️⃣ 快速估值分析模式（特殊）
```bash
--valuation 600519
```
- **说明**: 专门的估值分析工具
- **数据源**: 雪球API + 百度股市通
- **输出**:
  - 当前估值（PE/PB/PS/PCF/股息率）
  - 历史分位数（10年/5年/3年/1年）
  - DeepSeek深度估值分析
- **适用**: 快速估值判断
- **独立模式**: 不与 analysis-mode/prompt-version 组合

| 维度 | Professional | Value First | Quant Hybrid |
|------|-------------|-------------|--------------|
| **核心理念** | 专业严谨 | 价值是锚 | 多因子打分 |
| **输出格式** | 结构化报告 | 价值+择时框架 | 打分表+大白话 |
| **估值关注** | 综合评估 | **核心锚点** | 30%权重 |

| **语言风格** | 专业+通俗 | 大白话+比喻 | 极度通俗 |
| **适合人群** | 有经验投资者 | 价值投资者 | 所有人 |

### Professional（专业分析师版）

**输出示例**:
```markdown
### 综合评级
**中性偏空** | 置信度：65%

### 核心逻辑
公司当前基本面显著恶化，营收与利润双双大幅下滑，且现金流质量堪忧，
高估值缺乏业绩支撑；尽管短期技术面与资金流略有改善，但不足以扭转
中期下行趋势。

### 关键信号
1. 基本面恶化信号
2. 现金流与估值背离信号
3. 技术面弱势信号

### 操作策略
- 短线: 观望，仓位0%
- 中线: 减仓/等待
```

**特点**:
- ✅ 结构清晰，逻辑严密
- ✅ 适合机构投资者或有经验的个人投资者
- ✅ 平衡各维度分析
- ⚠️ 相对专业，新手可能需要理解时间

---

### Value First（价值优先版）

**输出示例**:
```markdown
### 综合评级
**中性偏空** | 置信度：65%

### 核心逻辑
当前估值（PE-TTM 55.25， PB 7.44）显著高于传统价值投资的安全边际，
且基本面出现营收与利润双降、现金流恶化的不利信号，价值锚定作用薄弱。
技术面虽短期有资金流入，但股价处于中长期均线压制下的"均线纠缠"状态，
缺乏明确上行趋势。

### 关键信号
1. **价值面信号**: 估值高企且质量恶化（核心问题）
2. **技术面信号**: 趋势不明，上方承压（择时判断）
3. **催化剂信号**: 资金流入与事件平淡（缺乏驱动）

### 操作策略
- 短线: 观望或谨慎试空（60元阻力位）
- 中线: 暂不建仓，等待PE降至40倍以下或业绩拐点
```

**特点**:
- ✅ **估值分析最深入**（安全边际、现金流质量）
- ✅ 技术分析作为择时工具（而非决策依据）
- ✅ 强调催化剂和业绩拐点
- ✅ 适合中长期投资者
- ⚠️ 对短线交易者参考价值有限

### 一句话结论
别买，太贵了，而且公司赚钱能力在变差。

### 关键观察点
1. **最弱因子 - 质量与估值**: 净利润下滑25%，PE却高达55倍
2. **最强因子 - 情绪**: 有资金在博弈，但撑不住基本面
3. **需关注的边际变化**: 下季度利润能不能止跌

### 操作建议
- 短线: 56元轻仓试多（逆势，快进快出）

- ✅ **最通俗易懂**（极度大白话）

- ✅ 用生活化比喻和具体金额举例
- ✅ 适合所有投资者（包括新手）
- ✅ 系统化决策，减少主观判断
- ⚠️ 打分模型较简化，复杂情况可能不够精细

## 🔧 智能触发机制详解

### 触发条件（auto模式）

当使用 `--analysis-mode auto` 时，系统会自动评估信号强度：

#### 评分系统
| 信号类型 | 触发条件 | 得分 |
|---------|---------|------|
| **资金流向** | 流入≥10亿 | 3分 |
|  | 流入5-10亿 | 2分 |
|  | 流入<5亿 | 1分 |
| **技术突破** | 突破年线MA250 | 3分 |
|  | 突破月线MA20 | 2分 |
|  | 接近关键位 | 1分 |
| **基本面** | 业绩爆发（增长>50%） | 3分 |
|  | 业绩改善（增长>20%） | 2分 |
| **波动率** | 单日涨跌>7% | 2分 |
|  | 单日涨跌>5% | 1分 |

**触发阈值**: 总分 ≥ 3分 → 自动启用Brain层深度分析

---

#### 场景A: 平静期（不触发）
```
- 资金: 流入3000万（0分）
- 技术: 月线附近横盘（0分）
- 业绩: 利润微增5%（0分）
- 波动: 当日涨1.2%（0分）
总分: 0分 → 使用快速模式
```

```
- 资金: 流入6亿（2分）

总分: 8分 → 🧠 触发深度分析
```

## 🔄 模式切换建议

### 从简单到复杂的学习路径

1. **新手阶段**: 使用 `quant_hybrid`
   - 量化打分最直观
   - 大白话解释最易懂

2. **进阶阶段**: 尝试 `professional`

3. **深度阶段**: 使用 `value_first`
