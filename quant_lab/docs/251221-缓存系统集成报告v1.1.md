## ⚠️ 已知问题

### 1. 部分指数代码可能无法获取数据

**现象**: H30533、1B0932 等指数代码在 AkShare 中可能没有对应数据

**解决方案**: 建议使用更通用的指数代码：
- `000300` - 沪深300
- `399006` - 创业板指
- `000016` - 上证50
- `399905` - 中证500

### 2. 部分估值数据可能获取失败

**现象**: 日志显示 "估值指标获取失败: AttributeError"

**原因**: AkShare API 接口变化或数据源问题

**影响**: 不影响系统运行，只是部分四维数据不完整

**解决方案**: 系统会优雅降级，使用可获取的数据
