## 📦 技术栈确认

- **后端**: Flask + SQLAlchemy
- **数据库**: MySQL（生产）/ SQLite（开发）
- **缓存**: Redis（可选）
- **支付**: PayJS（个人推荐）或 Ping++
- **前端**: HTML + Tailwind CSS + Alpine.js
- **服务器**: 阿里云ECS（2C4G起）
- **部署**: Nginx + Gunicorn
