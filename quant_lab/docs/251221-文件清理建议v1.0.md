## ✅ 建议清理（安全）

### 1. Python缓存目录
```bash
rm -rf __pycache__
```
**理由**: 运行时会自动重新生成，删除后不影响功能

### 2. macOS系统文件
```bash
rm -f .DS_Store
```
**理由**: macOS自动生成的文件夹显示设置，可随时重建

### 3. 旧文件归档
```bash
rm -rf archive/
```
**理由**: 包含旧版本代码和过时的测试文件
**内容**:
- `ai_analyst_*.py` - 旧版本分析器
- `test_news_*.py` - 新闻接口测试
- `diagnose_akshare.py` - 诊断脚本
- `trend_3years.py` - 旧版趋势分析

### 4. 测试文件
```bash
rm -f test_cache_integration.py
rm -f test_performance.py
rm -f quick_test.py
rm -f test_cache.db
```
**理由**: 缓存系统已集成完成，测试文件可以删除

### 5. 示例文件
```bash
rm -f main_enhanced_example.py
```
**理由**: 四维数据功能已集成到 `main.py`，示例文件不再需要

### 6. 旧版模块
```bash
rm -f analyst_brain.py
```
**理由**: 已被 `analyst_brain_v2.py` 替代

## ⚠️ 可选清理（谨慎）

### 7. 股票列表缓存
```bash
rm -f .stock_list_cache.json
```
**理由**: 352KB，可能包含过期数据
**影响**: 删除后首次查询会重新下载股票列表（约5-10秒）
**建议**: 如果查询速度正常，可以删除

### 8. 别名脚本
```bash

```
**理由**: 可能是临时使用的shell别名
**建议**: 先检查内容，确认不需要再删除

## 🚀 一键清理脚本

创建清理脚本 `cleanup_files.sh`:

```bash
#!/bin/bash

echo "开始清理 quant_lab 项目..."

# 1. Python缓存
echo "清理 Python 缓存..."
rm -rf __pycache__

# 2. 系统文件
echo "清理系统文件..."
rm -f .DS_Store

# 3. 旧文件归档
echo "清理 archive 目录..."
rm -rf archive

# 4. 测试文件
echo "清理测试文件..."
rm -f test_cache_integration.py
rm -f test_performance.py
rm -f quick_test.py
rm -f test_cache.db

# 5. 示例文件
echo "清理示例文件..."
rm -f main_enhanced_example.py

# 6. 旧版模块
echo "清理旧版模块..."
rm -f analyst_brain.py

# 统计
echo ""
echo "✅ 清理完成！"
echo "节省空间: 约 500KB"
echo ""
echo "可选清理（需要确认）:"
echo "- .stock_list_cache.json (352KB)"
echo "- aliases.sh (2.8KB)"
echo ""
echo "运行以下命令查看:"
echo "  ls -lh"
```

**使用方法**:
```bash
chmod +x cleanup_files.sh
./cleanup_files.sh
```
