# diagnose_data_sources.py

多数据源管理器，诊断和评估金融 API 的健康状况。

## 功能

- **DataSourceManager 类**: 实现智能多源降级 + 质量监控
- **估值数据获取**: 按优先级尝试东方财富 → 新浪财经，自动降级
- **数据完整度验证**: 检查 PE/PB/股息率等关键字段
- **成功率统计**: 记录各数据源的成功/失败次数

## 运行

```bash
cd /Users/hainingyu/Code/quant_lab
uv run python tests/diagnose_data_sources.py
```

默认测试 3 只股票（西部超导、贵州茅台、平安银行），输出各数据源成功率报告。

## 输出示例

```
📊 数据源质量报告
============================================================
✅ eastmoney_spot  : 100.0% (3/3)
✅ sina            : 100.0% (3/3)
============================================================
```

## 使用场景

- 大量股票估值显示 N/A 时，快速定位是哪个 API 故障
- 引入新数据源前评估稳定性
- 网络不稳定时检查 API 可用性
